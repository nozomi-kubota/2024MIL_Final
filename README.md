- 投薬記録入力画面で、以下を工夫

  - 30分単位でのデータ入力を行うため、JSを利用

## 難しかった点・次回トライしたいこと(又は機能)
-OURA RING API連携によるデータ表示が最後まで出来ず、
⇒ネットワークエラー的な表示が続くため、自社PC利用に起因する可能性あり。何とか実現したいです。
-チャート表示元をテーブルとして、テーブルから薬別平均効果持続時間をJSで算出する部分が上手く機能しないため、実装できるまでトライしたいです。
